Introduction

JavaScript is a cross-platform, object-oriented scripting language. It is a small and lightweight language.

It can be connected to the objects of its environment to provide programmatic control over them.

Hello world

To get started write your first "Hello world" JavaScript code:

function greetMe(yourName)
{
alert("Hello " + yourName);
}
greetMe("World");

Variables

The names of variables follow the next rules:

  • Start with:
    • letter
    • underscore
    • dollar sign
  • Can contain letters and digits
  • Javascript is case sensitive
Data types

There are seven data types:

  • Boolean true, false
  • null
  • undefined
  • Number 4, 6.12, 1.2e12
  • String "This is a string"
  • Symbol: is immutable and unique
  • Object

Objects are container which contain other variables. This variables can also be Objects
